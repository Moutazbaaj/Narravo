Overview

Narravo is a mobile application designed to convert text into speech, extract text from images, and scan documents for text recognition. It enhances accessibility, particularly for individuals with reading or speaking difficulties.

Features

1. User Authentication

  • Login: Users can sign in using their registered email and password.
  • Forgot Password: Users can reset their password by entering their email and receiving a reset link.
  • Sign Up: Users can create a new account with a valid email and a strong password (minimum 8 characters, including uppercase and lowercase letters and numbers).
  • Profile Creation: After signing up, users can select a profile picture, nickname, birthdate, and a color tag for customization.
  • Profile Editing: Users can modify their profile details through the settings menu.

2. Navigation & User Interface

  • Hamburger Menu: Provides quick access to user profile, settings, language selection, voice preferences, logout, and account deletion options.

3. Main Tabs

  • Home Tab: Displays shortcuts to key features like Text-to-Speech, Text from Photo, and PDF to Speech. Users can customize their view to show favorite collections.
  • Collections Tab: Allows users to create and manage text collections, add text manually or through extraction, and read text in individual or batch mode.
  • History Tab: Stores text interactions, including text-to-speech history, extracted text from photos, and PDF extractions. Users can delete history entries manually.

Privacy Policy "Data Collection"

Your privacy is important to us. Narravo uses Google Firebase Firestore to store user information, including your email, saved texts, collections, and history logs. We do not share your data with third parties.

Data Collection:

  • Data is only collected if the user creates an account.
  • Without an account, no user data is stored or collected.

Data Collected (for registered users only):

  • Email address (for authentication and account management)
  • Saved texts, collections, and history logs (for app functionality)

Data Storage: All user data is securely stored using Google Firebase Firestore with industry-standard encryption.

For more information on Firestore data privacy, visit: Firebase Privacy Policy

User Rights:

  • Users can request access to their data.
  • Users can delete their account, which permanently removes all associated data.

How to Delete Your Data: Navigate to the settings menu in the app and select the account deletion option.

Features Available Without an Account:

  • Text-to-speech conversion
  • Text extraction from images, camera, or PDFs
