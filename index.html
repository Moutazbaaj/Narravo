Overview

Narravo is a powerful mobile application designed to convert text into speech, extract text from images, and scan physical documents for text recognition. The app enhances accessibility, especially for individuals with reading or speaking difficulties, by providing an intuitive way to interact with text-based content.

Features

1. User Authentication

  • Login: Users can sign in using their registered email and password.
  • Forgot Password: Users can reset their password by entering their email and receiving a reset link.
  • Sign Up: Users can create a new account with a valid email and a strong password (minimum 8 characters, including uppercase and lowercase letters and numbers).
  • Profile Creation: After signing up, users can select a profile picture, nickname, birthdate, and a color tag for customization.
  • Profile Editing: Users can modify their profile details through the settings menu.

2. Navigation & User Interface

  • Hamburger Menu: Provides quick access to user profile, settings, language selection, voice preferences, logout, and account deletion options.

3. Main Tabs

  • Home Tab: Displays shortcuts to key features like Text-to-Speech, Text from Photo, and PDF to Speech. Users can customize their view to show favorite collections.
  • Collections Tab: Allows users to create and manage text collections, add text manually or through extraction, and read text in individual or batch mode.
  • History Tab: Stores text interactions, including text-to-speech history, extracted text from photos, and PDF extractions. Users can delete history entries manually.

Technologies Used

  • Frontend: SwiftUI (native iOS frameworks)
  • Text-to-Speech: AVSpeechSynthesizer (iOS-native API)
  • OCR: Vision Framework (iOS-native OCR technology)
  • Document Scanning: VisionKit (iOS-native document scanner)
  • Backend: CoreData and Firebase (for local and cloud storage of user data)

Account & Data Management

  • Logout: Users can log out at any time through the menu.
  • Delete Account: Users can permanently delete their account and associated data, including profile information, saved collections, and history logs.
